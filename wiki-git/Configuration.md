This page is divided into 2 parts :
* Parameter Table;
![image](https://user-images.githubusercontent.com/43994777/233339675-37205315-1ec4-4c83-ab03-dd7d6a200802.png)

* Telegram server settings.
![image](https://user-images.githubusercontent.com/43994777/233341789-6d345703-d6cd-41e7-a112-bdeb3d3fc4fb.png)

Parameter Table has the following columns:
* Parameter name;
* Parameter value;
* Default value;
* Actions:
    - Reset;
    - Accept/reject changes made while editing.

List of all parameters:
* BotName - Installed by BotFather botname. It must be ended with 'bot'. Need to insert here the name generated by BotFather;
* BotToken - Generated by BotFather secret access token. Need to insert here the token generated by BotFather;

The telegram section contains instructions for setting up a telegram bot, as well as the ability to restart a telegram bot, which is necessary when setting it up. This section is relevant only for the first configuration.
