# Ключи доступа

Ключи доступа — это окно, содержащее список всех ключей доступа на сайте.

---

## Структура

Ключи доступа по структуре очень похожи на окно Продукты. Страница состоит из списка ключей и строки поиска.

Поиск выполняется по ключу, можно использовать только часть ключа, после чего будет найдена вся информация по нему.

Список ключей состоит из 7 элементов:

* **Product** — имя продукта, к которому принадлежит этот ключ;
* **Name** — имя ключа, указанное пользователем при создании. Ключ Default создаётся при создании продукта;
* **Key** — сам ключ доступа;
* **Last update** — отображает время последнего обновления данных продукта;
* **Permissions** — позволяют редактировать или удалять продукт;
* **Author** — содержит информацию обо всех разрешениях, которые имеет ключ:
  * **CanSendSensorData** — пользователь сможет отправлять данные датчиков с помощью этого ключа;
  * **CanReadSensorData** — пользователь сможет читать данные датчиков с помощью этого ключа;
  * **CanAddNodes** — пользователь сможет создавать узлы для этого продукта;
  * **CanAddSensors** — пользователь сможет создавать датчики для этого продукта;
  * **Full** — включает все перечисленные выше разрешения.
* **State** — отображает состояние ключа. Доступны 3 статуса:
  * **Active (🟢)** — этот ключ доступен для работы в соответствии со своими разрешениями;
  * **Expired (⌛)** — срок действия этого ключа истёк;
  * **Blocked (🔴)** — ключ был вручную заблокирован.
* **Actions** — список действий, которые можно выполнить с ключом:
  * **Edit** — открывает форму редактирования ключа;
  * **Block/Unblock** — вручную блокирует ключ, операции соответствующих разрешений блокируются до тех пор, пока менеджер/администратор не разблокирует ключ;
  * **Delete** — удаляет ключ.

---

## Создание ключа

В этой форме пользователь указывает:

* **Display name** — отображаемое имя ключа;
* **Permissions** — все разрешения, которые имеет ключ (описание каждого было выше);
* **Expiration** — срок действия этого ключа (указывается при создании, не может быть изменено).
