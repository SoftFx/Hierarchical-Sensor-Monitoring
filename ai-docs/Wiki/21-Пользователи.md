# Пользователи

В этом окне администратор может просматривать список всех зарегистрированных пользователей, добавлять и редактировать существующих пользователей HSM.

Окно содержит таблицу, разделённую на две части:

* Добавление нового пользователя;
* Таблица уже существующих пользователей.

Вся таблица состоит из 5 столбцов:

* **Username** — логин, который задаётся пользователем/администратором при регистрации;
* **Password** — пароль, который задаётся пользователем/администратором при регистрации;
* **IsAdmin** — флажок, определяющий, является ли пользователь администратором;
* **Products** — список всех продуктов, к которым пользователь подключён как ProductManager/Viewer;
* **Action** — список действий, которые можно выполнить над пользователем:
  * Удалить пользователя;
  * Редактировать пользователя (в настоящее время возможно только добавление/удаление прав администратора);
  * Подтверждение/отклонение редактирования.

Для добавления пользователя администратору необходимо задать Username, Password, при необходимости назначить права администратора и нажать кнопку "+".

Список продуктов и роль (ProductManager/Viewer) определяется в рамках каждого проекта. Через окно пользователя это изменить нельзя.
