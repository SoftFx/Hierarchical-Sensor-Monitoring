# Configuration

This is the admin panel, which contains the settings that are necessary for the correct operation of HSM.

See also: [Server Configuration](Server-Configuration), [Telegram Setup](Telegram-Setup), [Site Structure](Site-structure)

---

![image](https://user-images.githubusercontent.com/43994777/233079570-8aed0a6f-0b9a-41c7-8848-fa6681bd6f9c.png)

This page is divided into 2 parts:

## Parameter Table

![image](https://user-images.githubusercontent.com/43994777/233339675-37205315-1ec4-4c83-ab03-dd7d6a200802.png)

The Parameter Table has the following columns:

| Column | Description |
|---|---|
| **Parameter name** | Name of the configuration parameter |
| **Parameter value** | Current value (editable) |
| **Default value** | Factory default value |
| **Actions** | Reset to default, Accept/Reject changes |

## Telegram Server Settings

![image](https://user-images.githubusercontent.com/43994777/233341789-6d345703-d6cd-41e7-a112-bdeb3d3fc4fb.png)

| Parameter | Description |
|---|---|
| **BotName** | Bot name created via BotFather. Must end with `bot` |
| **BotToken** | Secret access token generated by BotFather |

The Telegram section contains instructions for setting up a Telegram bot, as well as the ability to restart the bot, which is necessary when setting it up. This section is relevant only for the first configuration.

See also: [Telegram Setup](Telegram-Setup)
