# Быстрый старт

На этой странице показан минимальный код для запуска HSMDataCollector в .NET-приложении.

---

### 1. Установка пакета

```bash
dotnet add package HSMDataCollector
```

### 2. Запуск с настройками по умолчанию

```csharp
using HSMDataCollector.Core;

var collector = new DataCollector(new CollectorOptions
{
    ServerAddress = "https://your-hsm-server",
    AccessKey = "YOUR_ACCESS_KEY"  // из Продукты → Ключи доступа в веб-интерфейсе
});

// Добавить встроенные системные датчики (необязательно)
collector.Windows.AddSystemMonitoringSensors()
                 .AddProcessMonitoringSensors()
                 .AddCollectorMonitoringSensors();

await collector.Start();

// Отправить значение пользовательского датчика
var cpuSensor = collector.CreateDoubleSensor("MyApp/cpu_usage");
cpuSensor.AddValue(42.5);

// При завершении работы:
await collector.Stop();
```

### 3. Добавление логирования (необязательно)

```csharp
using HSMDataCollector.Logging;

var collector = new DataCollector(new CollectorOptions
{
    ServerAddress = "https://your-hsm-server",
    AccessKey = "YOUR_ACCESS_KEY",
    Module = "MyService"
})
.AddNLog(new LoggerOptions { WriteDebug = true });

collector.Windows.AddProcessMonitoringSensors()
                 .AddSystemMonitoringSensors()
                 .AddCollectorMonitoringSensors();

await collector.Start();
```

После `Start()` откройте веб-интерфейс — ваши датчики появятся в продукте, связанном с ключом доступа.
