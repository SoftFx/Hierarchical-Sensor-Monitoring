# Состояния DataCollector

---

## Состояния коллектора

| Статус | Переключение после... | Действия |
| :--- | :--- | :--- |
| **Starting** | вызова метода `Start` | • Синхронизация с сервером<br>• Создание сессии<br>• Запуск датчиков<br>• Инициализация таймеров<br>• Инициализация очереди данных |
| **Running** | завершения метода `Start` и пользовательской задачи (если есть) | Отправка пользовательских данных на сервер |
| **Stopping** | вызова метода `Stop` | • Отправка оставшихся значений<br>• Очистка очереди данных<br>• Закрытие сессии<br>• Остановка датчиков<br>• Остановка таймеров<br>• Остановка очереди данных |
| **Stopped** | завершения метода `Stop` и пользовательской задачи (если есть) | — |

---

## Карта состояний

```
Stopped ──[Start]──> Starting ──[завершение]──> Running ──[Stop]──> Stopping ──[завершение]──> Stopped
```
