# Глоссарий

**Продукт** — логическая группа, которая должна указывать на физический компьютер или группу компьютеров. Каждый продукт должен иметь уникальное имя. Каждому продукту присваивается уникальный ключ, который используется для отправки данных, соответствующих именно этому продукту. В клиентском приложении продукт указывает корень дерева.

**Датчик** — сущность, которая передаёт свои состояния серверу. Датчик определяется именем и путём в дереве продукта.

**Ключ продукта** — строка символов, которая выдаётся продукту в процессе добавления. Ключ продукта используется для передачи данных датчиков и идентификации продукта.

**Узел** — подгруппа для объединения датчиков, меньшая, чем продукт, и представленная как узел в визуальном дереве.

**Администратор (Admin)** — роль пользователя, означающая, что пользователь имеет полный доступ. Администратор может просматривать данные всех существующих продуктов, управлять всеми продуктами и иметь доступ к странице администратора, где можно редактировать параметры конфигурации.

**Менеджер продукта (Product Manager)** — роль, существующая внутри продукта. По сути предоставляет доступ к редактированию страницы продукта, где менеджер может добавлять и/или приглашать новых зрителей и менеджеров по электронной почте, а также добавлять дополнительные ключи продукта.

**Зритель продукта (Product Viewer)** — простейшая роль, позволяющая пользователю видеть продукт в дереве мониторинга и на странице продуктов, но не даёт доступа к редактированию страницы продукта.
