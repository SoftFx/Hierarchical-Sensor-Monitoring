# Регистрация

---

## Первое подключение

Для начала работы в системе мониторинга пользователю необходимо зарегистрироваться.

После нажатия кнопки "Register" откроется форма, в которой пользователь должен ввести логин и пароль. Этот логин и пароль будут использоваться при авторизации. Минимальная длина пароля — 8 символов. После ввода логина и пароля необходимо нажать кнопку "Submit" для создания пользователя.

В случае успешной регистрации пользователь будет авторизован под своим именем и паролем.

Это происходит потому, что пользователь ещё не добавлен ни в один продукт. Пользователь может иметь 3 роли на сайте:

* ProductViewer (Зритель продукта)
* ProductManager (Менеджер продукта)
* Admin (Администратор)

После получения соответствующей роли для одного из продуктов он отобразится в дереве.

---

## Роли

### ProductViewer

Простейшая роль, позволяющая пользователю видеть продукт в дереве мониторинга и на странице продуктов, но не дающая доступа к редактированию страницы продукта.

### ProductManager

Роль, существующая внутри продукта. Предоставляет доступ к редактированию страницы продукта, где менеджер может добавлять и/или приглашать новых зрителей и менеджеров по электронной почте, а также добавлять дополнительные ключи продукта.

### Admin

Роль пользователя, означающая, что пользователь имеет полный доступ. Администратор может просматривать данные всех существующих продуктов, управлять всеми продуктами и иметь доступ к странице администратора, где можно редактировать параметры конфигурации.
