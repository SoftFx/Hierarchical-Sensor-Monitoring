# Конфигурация

Это панель администратора, содержащая настройки, необходимые для корректной работы HSM.

---

Страница разделена на 2 части:

* Таблица параметров;
* Настройки Telegram-сервера.

---

## Таблица параметров

**Parameter Table** имеет следующие столбцы:

* **Parameter name** — имя параметра;
* **Parameter value** — значение параметра;
* **Default value** — значение по умолчанию;
* **Actions** — действия:
  * Reset — сброс;
  * Accept/reject — принять/отклонить внесённые изменения.

---

## Список всех параметров

| Параметр | Описание |
|----------|----------|
| **BotName** | Имя бота, заданное BotFather. Должно заканчиваться на `bot`. Необходимо вставить сюда имя, сгенерированное BotFather. |
| **BotToken** | Секретный токен доступа, сгенерированный BotFather. Необходимо вставить сюда токен, сгенерированный BotFather. |

---

## Секция Telegram

Секция Telegram содержит инструкции по настройке бота Telegram, а также возможность перезапуска бота, что необходимо при его настройке. Этот раздел актуален только для первоначальной настройки.
