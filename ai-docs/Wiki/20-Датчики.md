# Датчики

HSM-сервер создан для сбора данных из различных источников. В настоящее время поддерживается только HTTPS API.

---

## Продукты

После настройки и запуска серверного и клиентского приложений необходимо добавить хотя бы один продукт для начала работы. Для добавления нового продукта используйте окно Products в клиентском приложении. Вашему продукту будет присвоен уникальный ключ, который потребуется для отправки данных датчиков этого продукта.

---

## API-эндпоинты

Вы можете использовать веб-API, который работает на порту 44330. Есть разные способы:

1. Используйте специализированное ПО для тестирования API, например Postman. Вам нужно отправлять POST-запросы и добавлять заголовок Content-Type = "application/json".
2. Используйте любой язык программирования и отправьте запрос с заголовками, указанными выше, и шаблоном данных со страницы Датчики.
3. Вы можете использовать Swagger UI — простой и быстрый способ отправки данных. Swagger также предоставляет примеры данных. Swagger UI запущен по адресу `{url}:44330/api/swagger/index.html`, где `url` — адрес вашего сервера HSM (тот, который вы указываете как `address` в конфигурации клиента).
