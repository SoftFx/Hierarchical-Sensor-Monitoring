# Интеграция с Grafana

HSM поддерживает возможность интеграции своих данных с сервисом Grafana. Поддерживаемые типы данных можно посмотреть в [Поддерживаемые источники Grafana](18-Grafana-источники.md).

Для подключения HSM к Grafana необходимо выполнить следующие шаги:

1. Подготовка Grafana для работы с HSM (выполняется один раз на сервере);
2. Подготовка HSM для работы с Grafana;
3. Создание источника данных в Grafana;
4. Интеграция данных (создание панелей мониторинга).

---

## Подготовка Grafana для работы с HSM

Для подготовки Grafana к интеграции с HSM выполните следующие шаги:

1. Войдите в свою учётную запись Grafana;
2. Откройте окно "Connection" через меню Toggle;
3. Найдите "JSON" в разделе connection data;
4. Установите "JSON";

Эта операция выполняется 1 раз, при первой интеграции систем. На этом подготовка завершена.

---

## Подготовка HSM для работы с Grafana

Работа с данными HSM осуществляется через ключи доступа.

Существует два типа ключей:

1. **Master Key** — даёт доступ ко всем продуктам сервера;
2. **Product Key** — ключ конкретного продукта.

Master Key может быть создан администраторами и используется только для взаимодействия с Grafana. Для операций, не связанных с Grafana, этот ключ не действителен.

Помимо использования ключа, необходимо указать на сайте, какие именно данные должны отправляться. Для этого кликните правой кнопкой мыши на продукте/узле/датчике, данные которого вы хотите отобразить в Grafana, и отметьте "Enable Grafana".

Рядом с выбранным продуктом/узлом/датчиком появится значок Grafana, указывающий, что данные для этой сущности будут отправляться (значок может не отображаться, если визуализация значков отключена в фильтрах).

Теперь для визуализации данных необходимо скопировать ключ продукта или master key, данные которого вы хотите визуализировать в Grafana.

---

## Создание источника данных в Grafana

1. Нажмите "Data source" в меню Toggle.
2. Нажмите "Add new data source"
3. Выберите источник данных JSON.

4. Нажмите на появившийся вариант JSON;

5. Укажите URL-адрес, например — `https://hsm.dev.soft-fx.eu:44333` + `/grafana/jsondatasource`. Полный URL — `https://hsm.dev.soft-fx.eu:44333/grafana/jsondatasource`;
6. Установите флажок "Skip TLS Verify";
7. Нажмите "Add header";

8. Установите флажок "Skip TLS Verify";

9. Заполните Header — key и Value — используйте ключ доступа из HSM (Master или Product);

10. Нажмите "Save & test"

Готово!

---

## Интеграция данных

После настройки источника данных вы можете создавать панели мониторинга Grafana, используя данные HSM.
