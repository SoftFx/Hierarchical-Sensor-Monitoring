# Типы датчиков

---

## Простые датчики

* **Bool** — возможные значения: true / false
* **Integer** — целые числа
* **Double** — числа с плавающей запятой
* **String** — текст
* **Timespan** (v3.15.0) — длительность
* **Version** (v3.18.0) — номер версии (например, 1.2.3)

---

## Bar-датчики

* **IntegerBar** — бар, содержащий свойства int Min, Max, Mean, Last Value за определённый период времени
* **DoubleBar** — бар, содержащий свойства double Min, Max, Mean, Last Value за определённый период времени

---

## Продвинутые датчики

* **File** — любой файл, который можно преобразовать в байтовый поток
