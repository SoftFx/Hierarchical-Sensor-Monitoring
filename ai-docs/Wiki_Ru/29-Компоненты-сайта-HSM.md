# Компоненты сайта HSM

**Продукт (Product)** — логическая группа, в которой должен быть указан физический компьютер или группа компьютеров. Каждый продукт должен иметь уникальное имя. Каждому продукту присваивается уникальный ключ, который используется для отправки данных, специфичных для этого продукта. В клиентском приложении продукт задаёт корень дерева.

**Датчик (Sensor)** — сущность, передающая своё состояние на сервер. Датчик идентифицируется по имени и пути в дереве продукта.

**Узел (Node)** — подгруппа для объединения датчиков, которая меньше продукта и отображается как узел в визуальном дереве.

Пример: `.module` является узлом для датчиков `Collector version`, `Service alive`, `Version`, а также для узлов `Collector queue stats` и `Process dotnet`.
