# Конструктор оповещений

Блоки конструктора разделены на 2 части: **Условия** и **Действия**. Вы можете добавлять различные типы блоков, нажимая кнопку «плюс», или удалять ненужные блоки, нажимая кнопку «крестик».

---

## Блоки условий

Набор событий для выполнения определённых действий. Блок условия начинается с блока `If` и может включать 3 части:

1. **Свойство (Property)**
   * `Value` — выберите это, если хотите создать оповещение для значения датчика типа `Integer` или `Double`.
   * `Min, Max, Mean, LastValue` — выберите это, если хотите создать оповещение для минимального, максимального, среднего или последнего значения датчика типа `IntegerBar` или `DoubleBar`.
   * `Status` — выберите это, если хотите создать оповещение для статуса датчика (доступно для всех датчиков).

2. **Операция (Operation)**
   * Арифметические операции `>=`, `>`, `<`, `<=` (доступны только если `Property` = `Value`/`Min`/`Max`/`Mean`/`LastValue`).
   * Операции статуса `is changed`, `is 🔴 Error` или `is 🟢 OK` (доступны только если `Property` = `Status`).

3. **Цель (Target)** (числовое значение для сравнения, доступно только если `Property` = `Value`/`Min`/`Max`/`Mean`/`LastValue`)
   * Целое число (для датчиков `Integer` и `IntegerBar`).
   * Число с плавающей запятой (для датчиков `Double` и `DoubleBar`).

---

## Блоки действий

Набор событий, которые выполняются при выполнении определённых условий. Блок действий начинается с блока `then`. Доступны следующие действия:

* `Send notification` — выберите, если хотите получать уведомления в Telegram с пользовательским шаблоном комментария. Это обязательное действие, шаблон комментария также обязателен (значение по умолчанию: `$sensor $operation $target`).
* `Show icon` — выберите, если хотите установить значок в дереве и получать уведомление в Telegram с этим значком.
* `Set 🔴 Error status` — выберите, если хотите изменить статус датчика на `Error`.

---

## Специальные настройки

### Оповещение с временем жизни (Time to live)

Оповещение TTL имеет специальное условие с `Property = Inactivity period` и целевым параметром `Interval`. Чтобы задать TTL, нажмите ссылку `Add TTL` и выберите нужный интервал и действия. Если вы хотите установить `TTL = Never`, удалите оповещение TTL.
